运算符

  • 自增（++）或自减（–）不会改变变量本身的数据类型
  • 赋值运算符（+=、-=、%=、/=、%=）不会改变变量本身的数据类型
  • 当逻辑运算符的符号左边为false时，&继续执行右边的运算，而&&不再继续执行
  • 当逻辑运算符的符号左边为true时，|继续执行右边的运算，而||不再继续执行
  • 当三位运算符和if-else语句都可以使用时，优先选择三位运算符
  • 三位运算符要求左右两边属于同一类
