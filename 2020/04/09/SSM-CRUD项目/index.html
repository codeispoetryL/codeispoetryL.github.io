项目介绍

概述

SSM：Spring MVC + Spring + MyBatis
CRUD：create（创建） + retrieve（查询） + update（更新） + delete（删除）

功能

  • 分页
  • 数据校验：
    • jQuery前端校验 + JSR303后端校验
  • Ajax
  • Rest风格的URI；使用HTTP协议请求方式的动词，来表示对资源的操作GET(查询)、POST(新增)、PUT(修改)、DELETE(删除)

技术

  • 基础框架SSM
  • 数据库MySQL
  • 前端框架bootstrap
  • 项目的依赖管理maven
  • 分页pagehelper
  • 逆向工程MyBatis Generator
